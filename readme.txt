loop.py: loopt over de verschillende geoint's en wanneer je de code in comment toevoegd aan je programma schrijft je de valentie-elektron energie weg  


plot.py: plot de energieën van de valentie-elektronen samen met de coulomb-energie in functie van de internucleaire afstand  


chargeplot.py: plot het verloop van de lading van Na in functie van de nucleare afstand.
(Voor plot.py en chargeplot.py code aan uhf en/of rohf toevoegen vermeld in commentaar bovenaan plot.py en chargeplot.py voor het doorlopen van loop.py)
rhf.py: de RHF methode toepassen op de gegevens met een willekeurige lambda. toont ook aantal elektronen op elke atoom en welke type (alpha/beta) elektron op elk atoom.
